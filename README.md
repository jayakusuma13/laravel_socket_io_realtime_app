# laravel_socket_io_realtime_app
Very Basic Laravel + Socket Io  Realtime Task Manager List

# Requirements:
-Laravel
-Socketio
-laravel-echo-server

# Step:
1. composer install
2. npm install
3. npm run dev
4. php artisan migrate
5. php artisan serve
6. sudo service redis-server start
7. laravel-echo-server start
